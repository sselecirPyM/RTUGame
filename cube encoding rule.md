# RTU方块编码规则 1.0
## 总览
这是RTU游戏的方块编码规则。RTU游戏的方块以4字节编码，旨在提高应用程序性能。

现在这只是一份草稿，我们将会修改它。

## 规则
RTU游戏的方块编码以小端存储。

| 位 | 用途 |
| ---- | ----|
| 31 | 指示是否包含Unicode编码信息 |
| 26~30 | 24个旋转方向的一个 |
| 20~25(包含Unicode时) | 材质编码 |
| 0~19(包含Unicode时) | 用于编码Unicode字符。我们不编码用户定义区域。 |
| 8~25 | 不使用 |
| 0~7 | 编码方块。为0时表示空气。 |
